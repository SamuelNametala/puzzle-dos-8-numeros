...::: Descrição sobre o pluzze :::...

O puzzle dos 8 números é um jogo de tabuleiro quadrado dividido em 9 espaços, 8 ocupados com peças e o ultimo fica vazio.
O jogo funciona da seguinte maneira: após as peças do tabuleiro inicial serem embaralhadas, o jogador deve reorganizá-las obedecendo a regra do jogo.
A regra do jogo diz que o jogador só pode movimentar uma das peças por vez, que estejam ao lado do espaço vazio, para esse espaço vazio.

...::: Descrição sobre o projeto :::...

Projeto utilizando inteligência artificial para solucionar o problema do puzzle dos 8 números.
Este projeto utiliza a busca A* com opção de duas heurísticas (peças fora do lugar e distancia de manhattan) para comparação.
Após inicias o programa, o usuário deve embaralhar os números e em seguida, clicar em uma das opções (heurísticas) para que o sistema reorganize as peças obedecendo a regra do jogo.
